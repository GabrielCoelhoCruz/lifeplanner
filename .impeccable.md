# LifePlanner — Design Context

## Target Audience
Young professionals (20-35) in Brazil who want to organize multiple areas of life — work, personal projects, academia, fitness, diet — in one clean tool. Tech-savvy, mobile-first users who appreciate good design.

## Use Cases
- Create and manage projects across life areas
- Break projects into tasks with priorities and deadlines
- Track sub-items/checklists within tasks
- Switch between list and kanban views
- Quick search across all tasks
- Export/import data as JSON backup

## Brand Personality / Tone
**Calm productivity** — the app should feel like a quiet, well-organized workspace. No visual noise, no gamification, no anxiety-inducing red badges. Just clean structure that lets you think clearly.

## Aesthetic Direction
**Editorial minimalism** — inspired by Scandinavian design magazines. Typography-driven hierarchy, generous whitespace, surgical use of color only for functional indicators (priority, status). The interface disappears so the content speaks.

## Design Tokens
- **Font**: Geist Sans (body) + Geist Mono (data/dates)
- **Colors**: Warm neutrals (stone palette), accent rose (#E11D48), semantic colors for priority/status
- **Icons**: Phosphor Icons (regular weight)
- **Spacing**: 4px base scale
- **Radius**: 6/10/16/9999px

## Quality Bar
Flagship — this should feel like a premium product, not an MVP.
